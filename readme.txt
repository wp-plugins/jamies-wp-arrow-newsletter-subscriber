== Description ==

A Widget to add an Arrow newsletter subscription form .

This is a plugin which creates a widget for Q5 Media’s Arrow Email Marketing system to easily a the newsletter subscription form onto your WordPress site. Simply install the plugin and add the widget to the sidebar of your choice and enter the details to link it to your newsletter list with Q5 Media’s Arrow Newsletter System and away you go, expanding your database of email addresses of people interested in your product/service/knowledge.

Q5 Media's Arrow Email marketing system can be <a href="http://www.arrowemail.com/" target="_blank">found here</a> 

User ID = Given by Arrow Service - individual account of user -> REQUIRED
List ID = Given By Arrow - id of database to add emails to -> REQUIRED
Submit Value = enter submit, process or whatever you want the button to say - leave blank if using background image for button through external css
Thankyou Url = Customize a thankyou page and enter url here -> leave blank to go to default	
Error Url = Customize a error page and enter url here -> leave blank to go to default

If you have any requests for improving this plugin let me know <a href="http://www.martiniboy.co.uk/wordpress-plugins/arrow-newsletter-subscriber/" title="Wordpress developer" target="_blank">here</a>

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload arrow-newsletter folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Drag the widget to you desired siderbar
